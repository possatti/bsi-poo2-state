Descrição do padrão

Permite a um objeto alterar o seu comporamento quando o estado interno muda.

Para implementa-lo, encapsulamos o estado com o seu contexto. Conforme a estrutura de exemplo abaixo.

Explicação do exemplo

Nesse exemplo, tempos uma pizza, que pode estar em três diferentes estados: cooked (cozida), baked (assada), delivered (entregue). E quando chamamos os diferentes métodos da pizza, ela dá conta de mudar o seu estado atual.

No código, temos a classe PizzaState que é a classe base para os diferentes estados que temos para pizza, cujas implementações são: BakedPizzaState, CookedPizzaState e DeliveredState.
